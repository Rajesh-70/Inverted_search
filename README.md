# Inverted Search System

## Overview
The Inverted Search System is a C-based project developed to efficiently search words across multiple text files using Data Structures and File Handling techniques. The project creates an indexed database of words along with their file references and occurrence counts for fast searching and retrieval.

---

## Features
- Create database from multiple text files
- Search words efficiently
- Display word occurrence count
- Display file references for searched words
- Save database to file
- Update existing database
- Handle duplicate words

---

## Technologies Used
- C Programming
- Data Structures
- Linked Lists
- Hashing
- File Handling
- Dynamic Memory Allocation
- Pointers
